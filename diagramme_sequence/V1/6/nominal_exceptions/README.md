# UC6 — Consulter les statistiques (SSD)

## Scénario nominal
1. Le **Gestionnaire** demande des statistiques sur une période (mois/année, dates).
2. Le **Système** agrège les réservations passées et calcule :
   - chiffre d’affaires,
   - taux d’occupation,
   - utilisation des ressources/options.
3. Le **Système** renvoie les indicateurs au gestionnaire.
